z-carousel demo

Base

Here is the raw component, You can add anything inside, Each root element will be considered as a slide unless you use one of the defined slots

Navigation

You can add navigation buttons. When a navigation is possible, it can be clicked, otherwize, the button will be disabled.

By default, the navigation buttons are provided with html arrow entities. It is possible to customize the content of the navigation buttons.

Use slot="nav-prev" and slot="nav-next" slots to do so.

Counter

You can add a fraction counter, this will show the current slide number over the total slides number

Dots

You can show dots for every slide available on the carousel, each dot but the current one is clickable.

Infinit

It is possible to cycle threw the carousel with the infinit attribute.

step

You can decide the number of slides to go forward or backwards for each navigation action.

current slide index

It is possible to make the carousel initalize on a given slide index.

This attribute will be updated over the navigation actions, you can update it too from outside.

per page

You can decide to display multiple slides per page.

This feature is interesting used with the step option
